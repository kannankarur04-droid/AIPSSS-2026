# 🎓 AIPSSS - AI Powered Student Support System

AIPSSS என்பது மாணவர்களின் கற்றல் திறனை மேம்படுத்துவதற்காக உருவாக்கப்பட்ட ஒரு அறிவுசார் கல்வி உதவியாளர் (Intelligent Educational Assistant). இது கூகுளின் **Gemini AI** தொழில்நுட்பத்தைப் பயன்படுத்தி, மாணவர்களின் சந்தேகங்களுக்குத் துல்லியமான தீர்வுகளை வழங்குகிறது.

## 🌟 முக்கிய அம்சங்கள் (Key Features)

- **எளிமையான விளக்கங்கள்:** கடினமான பாடக் கருத்துக்களையும் (Concepts) மாணவர்களுக்குப் புரியும் வகையில் எளிய உதாரணங்களுடன் விளக்கும்.
- **கணித மற்றும் அறிவியல் சூத்திரங்கள்:** $E = mc^2$ மற்றும் $\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$ போன்ற சூத்திரங்களை LaTeX வடிவில் மிகத் தெளிவாகக் காட்டும்.
- **சுயப் பரிசோதனை (Self-Assessment):** ஒவ்வொரு விளக்கத்திற்குப் பிறகும், மாணவர்கள் தாங்கள் கற்றதைப் புரிந்துகொண்டார்களா என்பதைச் சோதிக்க 2 பயிற்சி கேள்விகளை (MCQs) தானாகவே உருவாக்கும்.
- **அதிவேக பதில்கள் (Streaming):** ஏஐ பதில்களை ஒவ்வொரு வார்த்தையாகத் திரையில் காட்டும் (Typing Effect), இது பயன்படுத்துவதற்கு எளிதாக இருக்கும்.

## 🛠️ பயன்படுத்தப்பட்ட தொழில்நுட்பங்கள் (Tech Stack)

- **Language:** Python
- **Framework:** Streamlit
- **AI Model:** Google Gemini 1.5 Flash
- **Hosting:** Streamlit Cloud

## 🚀 பயன்படுத்துவது எப்படி? (Usage)

1. உங்கள் இணைய உலாவியில் (Browser) AIPSSS லிங்க்கைத் திறக்கவும்.
2. உங்கள் சந்தேகத்தை "Ask your educational doubt..." என்ற பெட்டியில் டைப் செய்யவும்.
3. ஏஐ வழங்கும் விளக்கத்தைப் படித்துவிட்டு, இறுதியில் கேட்கப்படும் கேள்விகளுக்குப் பதிலளிக்க முயலவும்.
